# USD to JPY リアルタイム為替計算アプリ

本ドキュメントは、ドル建ての金額を入力し、リアルタイム為替レートで日本円に換算するWebアプリケーションの技術スタックおよび実装計画をまとめたものです。

## 技術スタックの選定

1. **コアフレームワーク**: **React (Vite)**
   - コンポーネントベースのUI構築に最適であり、状態管理（入力されたUSD金額・取得した為替レート）を分かりやすく管理できます。環境構築には高速なビルドツールであるViteを使用します。
2. **スタイリング**: **Vanilla CSS**
   - 外部ライブラリへの依存を減らし、最大限の柔軟性を持たせます。グラスモーフィズム（Glassmorphism）やカスタムダークモード、スムーズなマイクロインタラクションを取り入れた、プレミアムでリッチなUIを提供します。
3. **フォント・タイポグラフィ**: 
   - **Google Fonts (Inter または Outfit)** による、可読性が高くモダンなタイポグラフィをベースにします。
4. **為替レートAPI**: **Frankfurter API** または **ExchangeRate-API**
   - 無料で利用できるパブリックAPIを使用します。（FrankfurterはAPIキー不要でECBのレートを使用できるため、手軽にセキュアな実装が可能です）

## 実装計画

### フェーズ1: プロジェクト基盤の構築
- `Vite` を用いたReactプロジェクトのセットアップ。
- `index.css` にアプリ全体で利用するデザインシステム（カラーパレット、フォント、CSS変数）をインポートし、定義する。
- 適切なメタデータ、 `title`タグなど、SEOベストプラクティスを `index.html` に組み込む。

### フェーズ2: UIコンポーネントの設計と実装
プレミアムなユーザー体験を目指し、各パーツをコンポーネント化して構築します。
- `App` (メインコンテナ): 画面中央に配置される美しいカード型のUIコンテナ。
- `Header`: アプリ名と当日の日付等を表示。
- `CurrencyInput`: 動的なフォーカスアニメーションを備えたドル（USD）用の入力フィールド。
- `ExchangeResult`: 換算後の円（JPY）金額を大きく表示し、適用されている為替レート（1ドル=○○円）も併記する。

### フェーズ3: 為替API連携とロジック実装
- コンポーネントの初期マウント時に為替APIをフェッチし、現在のUSD/JPYレートを取得。
- 通信中のローディング状態（スケルトンUIやスピナーなど）、およびネットワークエラー時のフォールバック処理を実装。
- ユーザーの金額入力に合わせてリアルタイムで計算し、結果画面へ即座に反映させるステート管理（`useState`, `useEffect`）。

### フェーズ4: アニメーションとブラッシュアップ
- 各デバイス（スマートフォン・タブレット・PC）での表示を最適化するレスポンシブ対応。
- 金額が変化した時の数値のロールアニメーションや、ボタン等のホバーエフェクトの調整をVanilla CSSで行い、UI/UXを最高品質に仕上げる。
